# GIF_ASCII_CONVERTER

## 使い方

パッケージのインストール

```
python3 -m pip install pillow numpy opencv-python
```

GIF_ASCII_CONVERTER 内のディレクトリで

```
python3 test.py
```

pillow_imagedraw.gif を確認
